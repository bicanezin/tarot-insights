# **App Name**: Tarot Insights

## Core Features:

- Spread Display: Display a selected tarot spread layout, showing card positions and allowing card selection for each position. There will be one Yes/No spread (1 card), one Yes/No spread (3 cards), two love spreads, two professional spreads and two financial spreads.
- Card Interpretation: Display the name of the card, its image, and both upright and reversed meanings after a card is drawn. Allow toggling between upright and reversed interpretations.
- Reading History: Allow users to enter a name and date for each reading, saving this data locally along with the spread results for future reference. Show all details, including the position the card was played on the table, so they know why is the user considering about something to happen.
- AI Reading Assistant: Based on the selected spread and drawn cards, generate a personalized interpretation of the reading, considering the relationships between cards. Act as a tool for integrating details of the individual cards for the complete response. User can indicate details to be added on interpretation.
- Multilingual Support: Offer an i18n solution to have the website available in English and Portuguese (Brazilian version).

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke spirituality and intuition.
- Background color: Very light grey (#F5F5F5) to create a clean, neutral base.
- Accent color: Pale rose (#FFBCCD) for gentle highlights and calls to action.
- Elegant, serif fonts for card names and meanings. Clear, sans-serif fonts for other informational texts.
- Use symbolic icons, possibly inspired by traditional tarot imagery. Consider animated transitions for drawing cards.
- Ensure layouts for various screen sizes adapt fluidly, so that on larger screens there aren't unsightly expansions of content
- Animation: Subtle fade-in effects for card reveals, animated shuffling, and gentle glow effects on the selected cards
